A multilingual dialog-based tool to create SPC actors and Gift/Rite items for the World of Darkness 5e system.

This module focused on W5 that facilitates importing content directly from official book texts. Gifts, Rites, and NPCs (based on the SCS template) can be inserted as text blocks and transferred in a structured way.

Please note that unwanted line breaks and word splits must be manually removed during import. Likewise, when importing NPC data, any additional bracketed notes following abilities must be cleaned up manually.

---

Ein mehrsprachiges, dialogbasiertes Tool zur Erstellung von NSC sowie Gaben- und Riten für das World of Darkness 5e-System.

Ein Modul mit Fokus auf W5, das den Import von Inhalten direkt aus offiziellen Buchtexten erleichtert. Gaben, Riten und NSC (auf Basis des SCS-Templates) können als Textblöcke eingefügt und strukturiert übernommen werden. 

Dabei ist zu beachten, dass störende Zeilenumbrüche und Worttrennungen während des Imports manuell entfernt werden müssen. Ebenso müssen beim Import von NSC-Daten eventuelle Klammerzusätze hinter Fähigkeiten händisch bereinigt werden.
